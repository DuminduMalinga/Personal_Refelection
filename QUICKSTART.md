# 🚀 Quick Start Guide

## Run Your New Dashboard in 3 Steps!

### Step 1: Run the App
```
1. Open project in Android Studio
2. Click green ▶ Run button
3. Select emulator/device
4. Wait for app to launch
```

### Step 2: Create Account
```
1. Tap "Register"
2. Fill in:
   - Full Name: "Your Name"
   - Email: "test@example.com"
   - Username: "testuser"
   - Password: "test123"
   - Confirm Password: "test123"
3. Tap "Register"
4. ✨ Dashboard appears automatically!
```

### Step 3: Explore Dashboard
```
✓ See your personalized greeting
✓ View the three stat cards (showing 0s)
✓ Try clicking the FAB (+) button
✓ Tap "View My Goals" button
✓ Check bottom navigation
✓ Tap overflow menu (⋮) → Logout
✓ Close and reopen app → Still logged in!
```
